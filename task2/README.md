# calc-server #

### Technology ###
* [Node.js](https://nodejs.org/en/)

### Prerequisite ###
```
git clone https://github.com/prapansak/calc-server.git
npm install -g nodemon
npm install
```

### Development ###
```
npm run dev
```

### Build ###
```
npm run build
```
* Copy `files` and `.env` file into build directory.

## API Document ###
### Calculator - Plus
Use Postman to test APIs.

<kbd>POST</kbd>
```

```
